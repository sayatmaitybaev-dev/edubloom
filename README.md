# EduBloom 🌸

Образовательная платформа для эффективного взаимодействия преподавателя и учеников.

## Стек технологий

- **Frontend:** React + TypeScript + Vite + Tailwind CSS
- **Backend:** Node.js + Express + TypeScript
- **База данных:** SQLite (через Prisma ORM)
- **Авторизация:** JWT (email или номер телефона)

## Запуск

### 1. Установить зависимости

```bash
cd server && npm install
cd ../client && npm install
```

### 2. Запустить сервер (порт 4000)

```bash
cd server
npm run dev
```

### 3. Запустить клиент (порт 3000)

```bash
cd client
npm run dev
```

Открыть: http://localhost:3000

## Функционал

### Преподаватель
- Управление списком учеников
- Создание и редактирование расписания
- Добавление домашних заданий
- Выставление оценок и ведение успеваемости

### Ученик
- Просмотр расписания занятий
- Отслеживание домашних заданий с отметкой о выполнении
- Просмотр оценок и среднего балла
